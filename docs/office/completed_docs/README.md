# Office completed docs (care home records)

This folder stores completed/signed records managed by the care home in Care Home Office.

## Subfolders

- `registration_records/`
- `agreement_records/`

## Handling rules

- Store only final/issued records in this folder.
- Use clear filenames with date and care home identifier.
- Keep prior signed versions; do not overwrite historical records.
