# At-home onboarding script

This script helps introduce voicemailcare.com in at-home setups.

## Short introduction

"This service helps us share non-urgent voice messages in one simple place.
It is not a live chat service and is not for urgent or medical matters."

## What to explain

- Only the latest message is kept per direction/channel.
- Messages are not monitored in real time.
- Keep notes and files outside the app, using paper, computer files, or phone notes if helpful.
- Use the Life File Guide for suggested file names and headings.
- Use direct contact routes for urgent or medical matters.

## Setup reminders

- Confirm who is invited and linked.
- Keep local expectations calm and practical.
- Update main contact details where useful.
